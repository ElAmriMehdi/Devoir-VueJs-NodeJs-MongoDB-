## Prerequis:  
* NodeJs
* MongoDB

## steps to start the app: 
First import data to MongoDB: 

```
mongoimport --db test --collection restaurants --drop --file primer-dataset.json
```
Go to /server and install required packages: 

```
node npm insall
```
Launch the server
```
node serverCrudWithMongo.js
```
